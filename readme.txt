== Description ==
This plugin allows you to sync WooCommerce products from another WooCommerce site to your own store. You are required to register an account with [TheCartPress](https://app.thecartpress.com/) in order to obtain an activation key.

There are three types of account you can sign up for.

1. Reseller only - you are syncing products from another WooCommerce store only.
2. Host only - you are only providing products list for other WooCommerce store to sync to.
3. Both reseller and host - you provide products list and sync from another store at the same time.

Data to be synced including:

* product name & price
* product variations for variable products
* product categories and tags
* product images (optional)

You can also choose to automatically sync products using daily cron.

Regular account only able to sync up to 10 products. With premium account you can sync unlimited number of products.

== Installation ==
= Minimum Requirements =
* WooCommerce 3.0 or greater
* WordPress 4.9 or greater

= Prerequisite =
* Register an account with [TheCartPress](https://app.thecartpress.com/) and get your activation key.

= Automatic installation =
1. Log in to your WordPress dashboard.
2. Navigate to the Plugins menu, and click "Add New".
3. Search and locate 'TCP Products Sync' plugin.
4. Click 'Install Now', and WordPress will take it from there.

= Manual installation =
Manual installation method requires downloading the 'TCP Products Sync' plugin and uploading it to your web server via your FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Post installation setup = 
1. Log in to [TheCartPress](https://app.thecartpress.com/) dashboard.
2. Fill in the host and reseller domain.
3. Log in to your Wordpress dashboard.
4. Go to Settings > TCP Products Sync, enter your activation key, then click 'Activate'.
